# My dotfiles 

This is my simple config files, including bash, zsh, vim, tmux config, I upload them here so that I can sync between multiple devices.  

## Bash

### Alias

`alias ll='ls -alF'`  
`alias la='ls -A'`  
`alias l='ls -CF'`  
`alias mv='mv -i'`  
`alias rm='rm -i'`  
`alias cp='cp -i'`  
`alias ..='cd ..'`  
`alias ~='cd ~'`  
`alias gs='git status'`  
`alias g='/usr/bin/git --git-dir=$HOME/dotfiles/ --work-tree=$HOME'`
